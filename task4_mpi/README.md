# **Задание по MPI**

Индивидуальный номер: **51**

Задача: Реализовать простейшую программу, демонстрирующую полезность MPI-паттерна "**Группы процессов и подкоммуникаторы**. Разделение MPI_COMM_WORLD на подгруппы с помощью MPI_Comm_split / MPI_Group_*.". Использовать в массиве числа типа **double**.

Команда для сборки: `make`

Команда для запуска: `mpirun -np [число MPI-процессов] ./main`

Пример команды для запуска: `mpirun -np 9 ./main`

# **Описание программы**

Найти сумму, максимум и минимум заданного массива

Как используется паттерн **"Группы процессов и подкоммуникаторы"**? Каждая группа процессов считает свою функцию (MPI_SUM, MPI_MAX, MPI_MIN). Внутри каждой группы - обычный пример использования редукции массива для подсчёта суммы/максимума/минимума. Схема легко расширяется на бОльшее количество групп и соответствующих им функций
