# **Задание по интринсикам**

Индивидуальный номер: **51**

Задача:
Реализовать простейшую программу, демонстрирующую полезность инструкции `_mm256_mul_ps`

Команда для сборки: `make`

Команда для запуска: `./build/main [путь к файлу с исходным изображением]`

Пример команды для запуска: `./build/main images_input/sunset.jpg`

# **Описание программы**

Цветокоррекция изображения - умножаем каждый из каналов R,G,B изображения на соответствующие им коэффициенты. Заданы значения коэффициентов R=1.25; G=1.05; B=0.75 для получения эффекта "тёплый закат"

Как используется `_mm256_mul_ps`? Для перемножения сразу 8 значений. То есть три таких инструкции позволяют посчитать значения для 8 пикселей (так как 3 канала)

Получившееся у меня время работы программы:
- 4112 миллисекунд - последовательная версия
- 3150 миллисекунд - версия с использованием avx (ускорение на ~четверть)
