**Задание по библиотеке std::threads**

Индивидуальный номер: **51**

Задача:
- Найти первое число, превышающее предварительно заданное
- реализовать при помощи mutex (без condition_variable), возвращать из потоков значение через future/promise
- long int / long long int (64-bit)

Команда для сборки: `make`

Команда для запуска: `./build/main [число потоков]`

Если число потоков:
 - Равно 0 - последовательная версия программы
 - Равно положительному целому числу - параллельная версия с указанным числом потоков
 - Не указано - параллельная версия с дефолтным числом потоков
